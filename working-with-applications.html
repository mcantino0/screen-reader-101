Lesson 6: Working with Applications

Working with applications is easier than you might think! Just tap the Windows key and type the name of a program to quickly open any application. To get into an application's file menu, just tap Alt, then use arrow keys to move through the menus. To switch between open programs, hold Alt and press Tab, and use Alt and F4 to close any application.

Task Keyboard Command

Maximize window Windows key + up arrow
Snap window to left or right Windows key + left/right
Navigating the file menu Alt, arrow keys
Navigate file menu with shortcut keys Tap Alt, then enter the menu shortcut key. Shortcut keys are indicated with an underlined letter, and your screen reader will announce the underlined letter after reading the menu option.
Switching between open programs Alt + tab
Saving a file Ctrl + S, or use Alt to navigate the file menu
Closing a program Alt + F4

Navigating the Ribbon (Microsoft Apps)

In Microsoft applications (like Word, PowerPoint, and Excel), the file menu is called the "Ribbon". The Ribbon hosts a wide variety of options to help you make the most of these applications. Just like the file menu in other applications, you can tap Alt to get up into the Ribbon, and you can use the arrow keys to move through the options. If there is a certain feature that you utilize regularly, you may want to memorize the shortcut keys for that task. Your screen reader will announce these shortcut keys as you move through the available options in the Ribbon.
